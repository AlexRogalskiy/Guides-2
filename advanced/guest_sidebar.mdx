---
title: Guest Sidebar
description: "Learn how to limit what Guests see in their Sidebar."
---

Flatfile allows you to update your sidebar to hide/show certain elements via the Space component. [Learn more about Spaces](hello.com). 

<Note>This configuration only impacts Guests, not admins.</Note>

```ts
const metadata = {
  sidebarConfig: {
    showGuestInvite: true,
    showDataChecklist: false,
    showSidebar: true,
  }
}
```

<br/>

| Prop                  | Required | Type | Description |
| --------------------- | ------------   | ------------   | ------------   |
| `showGuestInvite`       |      `optional` | **boolean** | The Collaborators link allows anyone to add another Guest to the Space.     |
| `showDataChecklist`     |      `optional` | **boolean** | The data checklist shows all of the data needed for uploading to the primary Workbook.    |
| `showSidebar`           |      `optional` | **boolean** | The sidebar as a whole can be hidden.     |
