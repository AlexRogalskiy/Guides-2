Actions are incredibly powerful buttons added to the UI that [when clicked by end users] trigger <Tooltip tip="Learn about Jobs">[Jobs](../concepts/jobs)</Tooltip> 
that run within the Flatfile ecosystem. 

You can write your own actions using the [@flatfile/actions](npmjs.com) helper library or find a plugin that suits your need.

## Adding your first action 

In this example, we're going to use [@flatfile/plugin-action-webhook](npmjs.com). This plugin creates a primary action in a Workbook 
that allows users to submit data from this sheet to your API.

<a target="_blank" class="group px-4 py-1.5 relative inline-flex items-center rounded-full shadow-sm text-sm font-medium" 
href="https://replit.com/@AshleyMulligan/BogusFinancialAssembler#index.ts">
    <span class="absolute inset-0 bg-primary-dark rounded-full group-hover:opacity-[0.9]"></span>

```jsx
listener.use(webhookAction({
	name: 'Submit Data',
	type: 'primary',
	format: 'key-value',
	url: '[https://webhook.site](https://webhook.site/)'
}))
```
